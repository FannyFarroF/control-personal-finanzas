# Registro de Ingresos y Gastos 

Este proyecto es una aplicación simple de JavaScript que permite a los usuarios gestionar sus finanzas personales. Con esta aplicación, puedes crear registros de ingresos y gastos, listarlos, eliminarlos, y ver un resumen del total de ingresos, gastos y el saldo disponible.

## Características

- Crear registros de ingresos y gastos.
- Listar todos los registros creados.
- Eliminar registros individuales.
- Mostrar un resumen con el total de ingresos, gastos y el saldo actual.
